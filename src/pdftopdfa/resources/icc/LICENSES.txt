sGray.icc
---------
Version: sGrey-v2-micro (compact grayscale profile)
Based on: sRGB grayscale equivalent

From: Compact-ICC-Profiles by Clinton Ingram (saucecontrol)
Copyright: Released to the public domain

This profile is released under the CC0-1.0 license (Creative Commons Zero).
No restrictions on distribution and use to the extent allowed by law.

These minimal ICC profiles contain the minimum tags necessary to correctly
represent a color space and use custom packing to minimize file size.
Intended for embedding in image files where profile size is a consideration.

Source: https://github.com/saucecontrol/Compact-ICC-Profiles
